## Description
우리 어디서 만나?

와플로 와, 와플에서 정하자!


원하는 일정을 만들어서 가고 싶은 사람을 초대하세요. 함께 가고 싶은 장소를 아카이빙하고 링크, 메모, 좋아요를 통해 의견을 나누고 장소를 확정해요.

귀여운 와플 캐릭터들과 함께 약속을 만들고 장소 아카이빙 해볼까요?
